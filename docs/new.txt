# adding gfx namespace
# most classes became lower case

# system (old -> updated)
  - Vector2           -> gfx::vector2
  - Vector3           -> gfx::vector3
  - Clock             -> gfx::clock
  - RGB               -> gfx::rgb
  - RGBA              -> gfx::rgba
  - callback::key     -> gfx::callback::key
  - callback::error   -> gfx::callback::error

# system (new)
* interpolators (hpp + cpp)
  - gfx::lerp
  - gfx::clamp
  - gfx::cubic namespace for cubic ease functions
    + ease_in
    + ease_out
    + ease_inout

* box (hpp + cpp)

* task (hpp + cpp)
  - gfx::task<T>        (T is a position type ex: gfx::vector<float>)
  - gfx::task_queue<T>  (T is a task - or even derived from task - queue (it uses std::vector btw))
  - gfx::mid_task<T>    (T is a positon type and this class is derived from gfx::task)

NOTE: mid_task have an extra middle interpolation vector between start and end for curve motion.
